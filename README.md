# EffinDom FUI-RS Demo

Live demo of the EffinDom FUI-RS SDK: retained-mode canvas UI built with Rust
and WebAssembly.

**[View the demo](https://fui-rs-demo.effindom.dev/)**

**[Demo source code](./fui-rs-demo-source/)**

The built static site is served from the repository root. The complete demo
source is included separately so the deployment remains directly hostable.

## Repositories

| Repository | Purpose |
|---|---|
| **[fui-rs-demo](https://github.com/zion-sati/fui-rs-demo)** | This repository and live demo |
| **[fui-rs](https://github.com/zion-sati/fui-rs)** | Rust SDK and controls |
| **[EffinDOM](https://github.com/zion-sati/EffinDOM)** | Runtime, engine, and documentation |
| **[create-fui-rs-app](https://github.com/zion-sati/create-fui-rs-app)** | `npx` scaffolder CLI |
